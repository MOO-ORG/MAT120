
### 37. True or False

**Statement:**  
If $F(x)$ is an antiderivative of $f(x)$, then

$$\int f(x)\,dx = F(x)+C$$

### Answer: ✅ **True**

#### Explanation

An **antiderivative** of $f(x)$ is a function $F(x)$ such that

$$F'(x)=f(x).$$

The **indefinite integral** represents the family of _all_ antiderivatives of $f$. Since any two antiderivatives differ by a constant, we write

$$\int f(x)\,dx = F(x)+C,$$

where $C$ is an arbitrary constant called the **constant of integration**.

Therefore, the statement is **True**.

---

## 38. True or False

**Statement:** If $C$ denotes a constant of integration, the two formulas

$$\int \cos x\,dx=\sin x+C$$

and

$$\int \cos x\,dx=(\sin x+\pi)+C$$

are both correct equations.

### Answer: ✅ **True**

#### Explanation

We know

$$\frac{d}{dx}(\sin x)=\cos x.$$

Also,

$$\frac{d}{dx}(\sin x+\pi)=\cos x,$$

because the derivative of the constant $\pi$ is zero.

Notice that

$$(\sin x+\pi)+C  
=\sin x+(C+\pi).$$

Since $C+\pi$ is still just another arbitrary constant, both expressions describe exactly the same family of antiderivatives.

Hence, the statement is **True**.

---

## 39. True or False

**Statement:** The function

$$f(x)=e^{-x}+1$$

is a solution to the initial-value problem

$$\frac{dy}{dx}=-\frac{1}{e^x}, \qquad y(0)=1.$$

### Answer: ❌ **False**

#### Step 1: Check the differential equation

The given function is

$$y=e^{-x}+1.$$

Differentiate:

$$y'=\frac{d}{dx}(e^{-x}+1)  
=-e^{-x}.$$

Since

$$-\frac1{e^x}=-e^{-x},$$

the differential equation is satisfied.

#### Step 2: Check the initial condition

Evaluate at $x=0$:

$$y(0)=e^0+1=1+1=2.$$

But the initial condition requires

$$y(0)=1.$$

The condition fails.

Therefore, the statement is **False**.

---

## 40. True or False

**Statement:** Every integral curve of the slope field

$$\frac{dy}{dx}=\frac{1}{\sqrt{x^2+1}}$$

is the graph of an increasing function of $x$.

### Answer: ✅ **True**

#### Explanation

For every real number $x$,

$$x^2+1>0.$$

Therefore,

$$\sqrt{x^2+1}>0.$$

Hence,

$$\frac{dy}{dx}  
=\frac{1}{\sqrt{x^2+1}}  
>0  
\quad \text{for all }x.$$

A function whose derivative is positive everywhere is **increasing everywhere**.

Thus every integral curve of this slope field is increasing.

Therefore, the statement is **True**.

---

| Problem | Answer |
| --- | --- |
| 37 | ✅ True |
| 38 | ✅ True |
| 39 | ❌ False |
| 40 | ✅ True |
